# data_analyze Intro Video Design

## Style Prompt

一支 21 秒的项目收官介绍短片：像工程复盘里的精致片头，而不是商业广告。画面应该有数据仓库、SQL、Agent 编排和报告输出的质感；整体克制、清楚、有完成感。主视觉使用深色分析台背景、细网格、表格行、流程节点和轻微纸张质感，让观众感觉这是一个已经跑通的学习型数据分析 Agent MVP。

## Colors

| Role | Hex | Usage |
|---|---|---|
| Background | `#10130f` | 深色画布，偏墨绿，不使用纯黑 |
| Panel | `#f3ead7` | 报告、表格、卡片的暖纸色 |
| Foreground | `#f8f1df` | 深色背景上的主文字 |
| Accent | `#83d18b` | 成功路径、节点亮点、最终状态 |
| Signal | `#d4583f` | 路由分支、关键线条、强调点 |
| Muted | `#8c927f` | 辅助文字、网格、次级标签 |

## Typography

- Primary: `PingFang SC`, `Hiragino Sans GB`, `Inter`, system sans-serif
- Numeric / code accents: `SFMono-Regular`, `Menlo`, monospace

## Motion Rules

- 运动像数据流推进：横向扫描、节点点亮、表格行上升、报告展开。
- 入口保持 0.35s-0.65s，镜头之间轻微交叠。
- 不使用炫光大渐变、霓虹赛博风、过度 3D 或夸张粒子。

## What NOT to Do

- 不把项目包装成通用平台或成熟商业产品。
- 不使用紫蓝科技渐变作为主视觉。
- 不使用空泛的 AI 口号。
- 不把所有信息塞进相同卡片网格。
